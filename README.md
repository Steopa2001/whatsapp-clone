## WhatsApp Clone
Ho sviluppato un clone di WhatsApp utilizzando HTML, CSS e JavaScript, integrando anche un sistema di intelligenza artificiale capace di generare risposte automatiche ai messaggi. Il progetto offre le principali funzionalità di una chat moderna, con un’interfaccia ispirata a WhatsApp Web. Presta particolare attenzione alla struttura, all’organizzazione del codice e all’esperienza utente, garantendo inoltre una webapp completamente responsive, perfetta per tutte le dimensioni di schermo. L’obiettivo è mostrare la capacità di creare applicazioni interattive lato front-end e di integrare tecnologie avanzate come l’AI all’interno di un flusso di messaggistica real-time.
