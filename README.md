## WhatsApp Clone
Ho sviluppato un clone di WhatsApp utilizzando HTML, CSS e JavaScript, integrando un sistema di intelligenza artificiale capace di generare risposte automatiche ai messaggi.
Il progetto riproduce le funzionalità principali di una chat moderna, con un’interfaccia ispirata a WhatsApp Web e un’attenzione particolare alla struttura, all’organizzazione del codice e all’esperienza utente.
L’obiettivo è mostrare la capacità di creare applicazioni interattive lato front-end e di integrare tecnologie avanzate come l’AI all’interno di un flusso di messaggistica real-time.
